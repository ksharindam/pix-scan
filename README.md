# Pix-Scan

A PyQt frontend of scanimage utility for using with some multifunction printers  

### Dependencies
* python3-pyqt5  
* pyqt5-dev-tools  
* sane-utils  

### Install
Open terminal inside pix-scan directory. Then run...  
`pip3 install --user .`  

### Uninstall
`pip3 uninstall pix-scan`  

### Supported Printers  
* HP Deskjet series  
* Brother Inkjets
