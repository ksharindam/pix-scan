# Pix-Scan

A Qt4 frontend of scanimage utility for using with HP and Brother scanners or printers  

### Dependencies
* python3-pyqt4  
* sane-utils  

### Install
Open terminal and change directory to quikprint-master. Then run...  
`pip3 install --user .`  

### Uninstall
`pip3 uninstall pix-scan`
